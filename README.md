# CRUD NodeJS + MySQL<br>
Inicialização do front:
```
npm i
npm run dev
```
Inicialização do back:
```
cd controller
npm i
npm run dev
```
Certifique-se de que tenha o `nodemon` instalado globalmente `nodemon -v`, caso não:<br>
### OSX/Linux
```
sudo npm i nodemon -g
```
### Windows
```
npm i nodemon -g
```
